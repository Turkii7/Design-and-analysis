Airport Shortest Path Program - Java Version

Requirements:
- Java JDK
- Gson library (gson-2.10.1.jar)

Files:
- Airport.java: Airport class
- Dijkstra.java: Dijkstra's algorithm implementation
- Main.java: Entry point
- airports.json: Sample airport data
- gson-2.10.1.jar: JSON parsing library

How to Compile:
1. Place all files in one directory.
2. Compile with:
   javac -cp gson-2.10.1.jar *.java

How to Run:
java -cp .;gson-2.10.1.jar Main  (Windows)
java -cp .:gson-2.10.1.jar Main  (Mac/Linux)
